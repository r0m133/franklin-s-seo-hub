## Instructions de téléchargement

1. Créer les dossiers dans `public/` :
   ```bash
   mkdir -p public/logos public/testimonials public/process public/blog
   ```

2. Télécharger toutes les images depuis le site askingfranklin.com réel (pas le domaine local)

3. Optimiser les images si nécessaire (compression, format WebP, etc.)

## Note importante
Les URLs ci-dessus utilisent le domaine local `asking-franklin-vfinal.local`.
Pour télécharger les images, il faudra accéder au site réel askingfranklin.com ou avoir accès aux fichiers originaux.
